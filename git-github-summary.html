Git

Git merupakan tool version control system yang open source yang terdistribusi untuk mengelola perubahan file di dalam folder, foldernya disebuh repository/repo. Riwayat perubahan file disimpan menggunakan serangkaian commit

Catatan:

Version Control System adalah sebuah sistem yang menyimpan snapshot (rekaman) perubahan pada source code

GitHub

GitHub adalah sebuah layanan berbasis cloud untuk menyimpan dan mengelola repository berbasis git
